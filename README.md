# Random Sampo (ランダム散歩)

Random Sampo は、東京の駅をランダムに選んで新しい場所を発見するためのウェブアプリケーションです。

## 現在の機能

- 東京の駅をランダムに選択（地下鉄とJR線を含む）
- 選択された駅の詳細情報を表示
  - 駅名
  - 路線情報
  - Google Maps上での位置表示
- 駅周辺のおすすめスポットを表示（観光スポット、カフェ、レストラン）
- 駅への訪問を記録
  - 訪問日
  - 天気
  - メモ
- 訪問済みの駅リストを表示
- 訪問済みの駅を選択対象から除外
