# Random Sampo (ランダム散歩)

Random Sampo は、東京の駅をランダムに選んで新しい場所を発見するためのウェブアプリケーションです。

## 機能

- 東京の駅をランダムに選択（まだ千代田線のみ）
- 選択された駅の詳細情報を表示
- Google Maps を使用して駅の位置を表示
- 駅周辺のおすすめスポット（観光地、カフェ、レストラン）を表示
- 訪問記録機能（まだ）
